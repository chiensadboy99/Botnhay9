# 🎲 VIP Tài Xỉu Prediction Server

Server quản lý key và xác thực cho công cụ dự đoán Tài/Xỉu PRO.

## 📋 Yêu cầu hệ thống

- Node.js >= 14.0.0
- npm >= 6.0.0

## 🚀 Cài đặt

```bash
# Clone hoặc tải project
cd vip-tai-xiu-server

# Cài đặt dependencies
npm install

# Tạo thư mục data
mkdir data

# Chạy server
npm start

# Hoặc chạy với nodemon (auto restart khi code thay đổi)
npm run dev
